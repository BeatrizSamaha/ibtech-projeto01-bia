## O projeto

Cartão de visita pessoal feito em HTML, CSS e JavaScript puros. Sem framework, sem dependência externa.

O site tem estética editorial bold — rosa, preto e creme — com efeitos interativos inspirados no site do Lando Norris.

---

## Stack

- **HTML5** semântico com Open Graph e meta tags
- **CSS3** — variáveis, Grid, Flexbox, animações, flip cards, bento grid, efeito glitch, shimmer, marquee
- **JavaScript** vanilla — cursor customizado, blob reativo, efeito magnético, text scramble, IntersectionObserver, Clipboard API

Sem Bootstrap, Tailwind, jQuery, React ou qualquer dependência. Zero.

---

## Interações JavaScript

- Cursor rosa customizado com blob que segue o mouse com delay
- Efeito magnético nos links de contato e navegação
- Text scramble no subtítulo do hero ao hover
- Faixa de texto correndo (marquee) entre seções
- Efeito glitch no nome ao hover
- Animação de entrada com IntersectionObserver — cada seção entra uma vez ao aparecer no viewport
- Flip cards na seção de hobbies
- Copiar e-mail via Clipboard API com feedback visual
- Link de WhatsApp direto no contato
- Nav com link ativo destacado conforme scroll
